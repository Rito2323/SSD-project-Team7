# SURVEY BUILDER PROJECT

## BUILD AND RUN
 
 - Run DB (Mongo DB)

 - Build and Run Backend\
        -> cd Mongo-rest\
        -> npm install \
        -> npm run build\
        -> node DB.js

 - Build and Run Frontend\
        -> cd survey-builder-frontend\
        -> npm install\
        -> npm run build\
        -> npm start

Frontend port : 3001
Backend port : 3000
